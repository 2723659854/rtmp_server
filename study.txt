安装ffmpeg软件
apk add ffmpeg

ffmpeg -i rtmp://yourserver/live/streamName -c copy -f hls -hls_list_size 0 -hls_time 60 /path/to/hls/output/streamName.m3u8
ffmpeg -i rtmp://127.0.0.1/a/b -c copy -f hls -hls_list_size 0 -hls_time 60 ./hls/b.m3u8
下面这个命令生成hls格式文件可以使用VLC播放的
ffmpeg -i rtmp://127.0.0.1/a/b -c:v h264 -c:a aac -f hls -hls_time 60 -hls_list_size 0  out ./hls/put.m3u8
